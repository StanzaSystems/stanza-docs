# JavaScript

Stanza has several available JavaScript libraries, for both client and server side applications.

## UI Toolkit Libraries for Browser/Client Side Usage
UI Toolkit libraries help client developers to build adaptive UIs that gracefully degrade with system stress, and send end users helpful messages.
Stanza offers JavaScript UI Toolkit libraries for:
- Browser
- React
